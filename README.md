# Hospital-Management-System-Analysis
Hospital Management System Analysis project using SQL and Power BI with 10K+ hospital records (2023–2025). The dashboard analyzes patient statistics, doctor specialization, disease trends, and billing insights through interactive KPIs, charts, and slicers to support healthcare decision-making.

Tools & Technologies

SQL – Data extraction, filtering, and analysis

Power BI – Data visualization and dashboard development

Excel / CSV Dataset – Data source

Dashboard Features
Key KPIs

The dashboard contains 6 key performance indicators:

Total Patients

Average Age

Total Doctors

Total Billing Amount

Amount Paid

Amount Pending

These KPIs provide a quick overview of hospital operations and financial performance.

Visualizations

The dashboard includes the following visualizations:

Bar Chart: Doctor specialization distribution

Donut Chart: Total bill amount by payment mode

Line Chart: Disease-wise patient count trend

Ribbon Chart: State-wise comparison of total amount, received amount, and pending amount

Interactive Filters (Slicers)

Users can filter the dashboard using:

Doctor Name

State

City

Registration Date

These slicers allow users to explore the data dynamically.

Navigation Feature

A Home button is added to improve dashboard navigation and allow users to easily return to the main dashboard.

Dataset Information

Total Records: 10,000+

Data Period: June 2023 – June 2025

Coverage: Hospitals across multiple states in India

The dataset includes information such as patient details, doctor specialization, disease diagnosis, billing amount, payment mode, and patient location.

Project Outcome

This dashboard helps hospitals:

Monitor patient statistics

Analyze disease trends

Track payment performance

Understand doctor specialization distribution

Improve data-driven decision making
